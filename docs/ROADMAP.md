# 🎯 Tally App - Development Roadmap

## App Overview
**Name:** Tally  
**Tagline:** "Smart Bill & Debt Tracking"  
**Target:** iOS & Android App Stores  
**Model:** Freemium

---

## 📱 Freemium Structure

### FREE Tier
- Track up to **10 bills**
- Track up to **3 debts**
- Basic stats & dashboard
- Manual data entry
- Local storage only
- Standard theme

### PRO Tier ($2.99/month or $24.99/year)
- ✨ **Unlimited bills & debts**
- 📊 **Advanced analytics** (spending trends, charts)
- ☁️ **Cloud sync** across devices
- 📤 **Export to PDF/CSV**
- 🔔 **Bill reminders** (push notifications)
- 🎨 **Premium themes** (4+ color schemes)
- 📸 **Receipt scanning** (future)
- 🔒 **Biometric lock** (Face ID/Touch ID)
- 💎 **Priority support**

---

## 🛠️ Tech Stack

### Frontend
- React 18 (existing)
- Capacitor 5 (native wrapper)
- Chart.js (data visualization)
- Local Storage (free tier)

### Backend (Pro Tier)
- **Firebase** or **Supabase** (cloud sync)
- **Stripe** (payment processing)
- **Push notifications** (FCM/APNS)

### Development Tools
- **Xcode** (iOS builds)
- **Android Studio** (Android builds)
- **Capacitor CLI** (build management)

## 🔐 Privacy & Security

### Data Collection (Free)
- Local storage only
- No data leaves device
- No analytics (optional)

### Data Collection (Pro)
- Email (for account)
- Bills/debt data (encrypted)
- Usage analytics (anonymous)
- Crash reports

### Privacy Policy Points
- Data encryption in transit (SSL)
- Data encryption at rest
- No selling of user data
- User can export/delete all data
- GDPR compliant
- CCPA compliant
