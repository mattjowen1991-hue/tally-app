### Free Tier
- ✅ Track up to 10 bills
- ✅ Track up to 3 debts  
- ✅ Beautiful dashboard with stats
- ✅ Category organization
- ✅ Payment tracking
- ✅ Debt payoff calculator
- ✅ Local storage (private & secure)

## 🎯 Why Tally?

- **Beautiful Design** - Premium glass-morphism UI that's a pleasure to use
- **Privacy First** - Your data stays on your device (free tier) or encrypted (premium)
- **No Bank Connection** - Simple manual tracking, no security risks
- **Debt Tools** - Smart payoff calculator shows exactly when you'll be debt-free
- **Actually Works** - Built by someone frustrated with existing apps

## 🛠️ Tech Stack

- **Frontend:** React 18
- **Native:** Capacitor 5
- **Charts:** Chart.js
- **Storage:** LocalStorage (Free), Firebase (Pro)
- **Payments:** Stripe
- **Push Notifications:** FCM/APNS

## 🚀 Getting Started

### For Users
1. Download from App Store or Google Play
2. Open app and complete onboarding
3. Start tracking your bills!
4. Upgrade to Pro for unlimited tracking

### For Developers
See [SETUP.md](docs/SETUP.md) for full development instructions.

```bash
# Install dependencies
npm install

# Add platforms
npm run cap:add:ios
npm run cap:add:android

# Open in IDE
npm run cap:open:ios
npm run cap:open:android
```
